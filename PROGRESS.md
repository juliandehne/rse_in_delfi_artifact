## Goal

Self-contained **artifact / replication package** for the DELFI 2026 paper
*"Research Software Engineering in the Learning Technologies: a replication study
of research paradigms in the DELFI community"*.

It must run for **artifact-track reviewers** who do **NOT** have:
- the proprietary **paper PDF folder** (full texts of the analyzed DELFI papers), and
- a **SAIA / KISSKI API token** for the LLM experiments.

So the pipeline ships all **intermediate products** in `data/` and, by default,
**skips** the two steps that need those unavailable resources.

## Source

Everything is adapted (read-only) from the **`main` branch** of the sibling submodule
`publications/rse-elearning-evaluation`
(remote `git@github.com:FabiLochner/rse-elearning-evaluation.git`).
We never write to that repo. Files are pulled with `git show origin/main:<path>`.

## "No full texts" guarantee

The analyzed DELFI paper **full texts never enter `data/`**. Verified: the experiment
result CSVs and all aggregations contain only **bibliographic metadata** (`id, title,
authors, year, start_page, end_page, doi, isbn, issn, …`), **label values**, and
**`_erklärung`** (LLM justification) columns — no `text`/`main_content`/`references`
column. The preprocessing output that *does* hold full texts is intentionally NOT shipped.
`run_pipeline.py --assert-no-fulltext` (run automatically) scans every CSV in `data/`
and fails if a banned text column appears or any cell exceeds the length threshold.

## Decisions

- Steps are **runnable `.py`** (not notebooks) so the management script can chain them.
- The human-annotation notebook (ipywidgets/Jupyter) is rewritten as a **console** script:
  non-interactive subset computation always runs; the interactive labeling loop only runs
  with `--paper-folder` (hence default skip). The shipped `df_human_reannotation_results.csv`
  is the authoritative human output.
- `--paper-folder` is a **parameter** (default `None`). Steps 1 & 4 require it; if absent and
  not skipped, they error with a clear message.
- LNI LaTeX modeled on `publications/pub_rse_methodology` (quarto-lni-template.tex / samplepaper.tex).
